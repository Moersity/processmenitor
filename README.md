### processmenitor -- a system thread menitor 
## dependency
* flask
* psutil
* sqlite3

## how to use
* initial db:  python dbhelper.py (delete the comment #  initialTable()) to create table
* startservices python processmenitor.py
* see it at 127.0.0.1:5000
